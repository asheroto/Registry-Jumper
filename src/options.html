How to configure (60 second setup):

To complete the setup, run the install.cmd script in the extension's folder. This will download regjump.exe to %LOCALAPPDATA%\Registry Jumper so that the extension can communicate with it.

Click the path below to copy it to your clipboard. Next, press Win+R to open the Run dialog, use Ctrl+V to paste the path, and then press Enter to run the install script.

The install.cmd script is located in the following path:

  • Chrome (Windows): %LOCALAPPDATA%\Google\Chrome\User Data\Default\Extensions\\_0\host\install.cmd
  • Edge (Windows): %LOCALAPPDATA%\Microsoft\Edge\User Data\Default\Extensions\\_0\host\install.cmd
  • Brave (Windows): %LOCALAPPDATA%\BraveSoftware\Brave-Browser\User Data\Default\Extensions\\_0\host\install.cmd
  • Opera (Windows): %LOCALAPPDATA%\Opera Software\Opera Stable\User Data\Default\Extensions\\_0\host\install.cmd

If the path can't be found, you may be using a non-default profile. To find your profile path, open chrome://version (this works for Edge, Brave, and others) and copy the Profile Path by selecting it and pressing Ctrl+C. Next, press Win+R to open the Run dialog, paste the profile path, then append it with \\_0\host\install.cmd, and press Enter. That should run the install script, but if it doesn't, find the extension path in the Extensions folder and append it with host\install.cmd.

Once you've completed the installation, press the Verify button to ensure everything is set up correctly!
